ShareGoal

แอป  ShareGoal  เป็นแอปที่คุณสามารถแบ่งปันเรื่องราวของคุณ  ไม่ว่าจะเป็นความฝัน  เป้าหมาย  หรือภารกิจให้ผู้อื่นอ่านหรือแลกเปลี่ยนความคิดเห็นและช่วยแนะนำก็ได้เช่นกัน  มาเริ่มเขียนเป้าหมายของคุณในพื้นที่ที่พร้อมจะสนับสนุนคุณกันเลย !

Genre : UX/UI Design

Program : Figma

Size : 72 x 147 mm (iPhone 13&14)

Result

Idea and Reference

Progress

1.  ทำการตั้งโจทย์ขึ้นมา โดยใช้เว็บ UI Coach

2.  กำหนดองค์ประกอบ ขอบเขตของงานและตั้งกลุ่มเป้าหมาย

3.  รวบรวมข้อมูลจากการทดลองใช้แอปที่เกี่ยวข้องและสนใจในการดีไซน์

4.  ร่างโครงร่างหน้าตาแต่ละหน้า

5.  หน้า Home เราจะออกแบบ Header ก่อน แล้วนำ search icon มาใส่

6.  การสร้าง Search bar

    6.1  สร้าง frame ขึ้นมา ทำเป็น Search box
    6.2  สร้าง เส้น แล้วใส่ component ลงไป
    6.3  สร้าง variant 2 ตัว ตั้ง state เป็น show, invisible
    6.4  state invisible เปลี่ยน opacity ให้เป็น 0% (ใส)
    6.5  เชื่อมทั้งสอง state เข้าด้วยกัน แล้วใส่ delay ตามเวลาที่ต้องการ
    6.6  ทำการ copy เส้นที่ได้ใส่ลงไปใส่ Search box ก็เป็นอันเสร็จสิ้น

7.  สร้าง Navigation bar ด้านล่าง

8.  สร้าง frame เป็นส่วนของ body

9.  สร้าง frame เก็บรูปโปรไฟล์ ชื่อ และข้อมูลสถิติส่วนตัวไว้ แล้วใส่ auto layout

10.  สร้าง frame เก็บสถิติความคืบหน้าของภารกิจและภารกิจที่บันทึกไว้

11.  สร้าง frame ทำเป็น Tag bar เป็นประเภทภารกิจ ใช้ auto layout แบ่งช่องไฟ

12.  สร้างกล่องเพิ่ม tag

13.  สร้างปุ่มเพิ่มโพสต์/ภารกิจ

14.  สร้าง Header ซึ่งจะมี ปุ่มออก และปุ่มโพสต์อยู่ 2 ด้าน

15.  สร้าง body ขึ้นมาเป็น frame

16.  สร้าง frame เก็บรูปโปรไฟล์ ชื่อ ความเป็นส่วนตัว และ tag

17.  การสร้าง private menu / tag menu

    17.1  สร้าง frame ขึ้นมา ทำเป็นกล่อง
    17.2  สร้าง text ใส่กล่อง
    17.3  เพิ่ม component จากนั้นก็สร้าง variant ตามข้อมูลที่ต้องการจะใส่
    17.4  เพิ่ม state hover แล้ว copy variant ที่ได้ มาเปลี่ยนสี
    17.5  เชื่อม state default กับ hover
    17.6  แล้วเพิ่ม animation ease in and out เมื่อ hover ลงไป

18.  สร้าง Footer ซึ่งเป็นข้อมูลเพิ่มเติมในการโพสต์

19.  สร้าง body ขึ้นมาเป็น frame

20.  การสร้าง item

    20.1  สร้าง frame ขึ้นมา ทำเป็นกล่อง
    20.2  สร้างรูปโปรไฟล์ ชื่อ และข้อความให้อยู่ใน frame เดียวกัน
    20.3  เพิ่ม auto layout ให้ส่วนนี้อยู่ด้านซ้ายบนตลอด
    20.4  สร้างปุ่มที่มีปฏิสัมพันธ์กับผู้ใช้
    20.5  เพิ่ม auto layout ให้กับปุ่มแต่ละส่วนให้อยู่ต่างตำแหน่งกัน
    20.6  ใส่ component แล้วสร้าง variant เป็น 3 สถานะคือ ปกติ, เลื่อน, ลบ
    20.7  สร้าง frame เบื้องหลัง เป็นสถานะเลื่อน
    20.8  เพิ่ม icon ระบบต่าง ๆ ลงไป แล้วใส่ auto layout
    20.9  เชื่อมสถานะปกติกับเลื่อน เมื่อถูกเลื่อนจะเข้าสู่สถานะเลื่อน item ไปทางซ้าย
    20.10  สร้างสถานะลบ โดยจะลดขนาดของสถานะเลื่อนลง
    20.11  ปรับ opacity 0% แล้วใส่ animation เพิ่มความสวยงาม
    20.12  เชื่อมปุ่มลบในสถานะเลื่อนและสถานะลบ

21.  ใส่ auto layout เข้าไปใน body

22.  สร้าง Header โดยมีเพียงปุ่มออก

23.  สร้าง body ขึ้นมาเป็น frame

24.  สร้าง frame เก็บข้อมูลโพสต์ โดยใช้ auto layout แบ่งช่องไฟของแต่ละส่วน

25.  สร้าง frame เก็บหัวข้อและคอมเม้น

26.  สร้าง frame เป็นคอมเม้นข้อมูล

27.  สร้างช่องเขียนคอมเม้น

28.  สร้าง body ขึ้นมาเป็น frame

29.  สร้าง item โดยใช้ auto layout จัดรูปและตัวหนังสือ แล้วเพิ่ม component

30.  สร้าง frame ในแต่ละหัวข้อแล้วใส่ item ลงไป

31.  ใส่ auto layout ช่วยจัดข้อมูลต่าง ๆ ข้างใน

32.  ( หาก item ใดก็ตามที่มีรูปร่างเหมือนกัน จะใช้ component ทั้งหมด )
