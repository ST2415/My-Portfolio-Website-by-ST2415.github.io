Blooko

หนังสืออ่านจนเบื่อแล้ว  จะทิ้งก็นึกเสียดาย  เว็บไซต์  Blooko  เปิดพื้นที่ให้ทุกคนสามารถเข้ามาแลกเปลี่ยนหนังสือได้  โดยจะเปิดโอกาสให้ผู้รับและผู้ส่งได้พูดคุย  ตกลงรับหนังสือได้ตามสะดวกไม่ว่าจะเป็นการนัดเจอกันหรือส่งไปทางไปรษณีย์

Genre : UX/UI Design

Program : Figma

Size : 72 x 147 mm (iPhone 13&14), 36 x 25 cm (Mac Pro 16)

Idea and Reference

Progress

(ไม่ได้เก็บภาพระหว่างการทำงานไว้ค่ะ)

1.  ทำการตั้งโจทย์ขึ้นมาก่อน โดยใช้เว็บ UI Coach

2.  กำหนดองค์ประกอบ ขอบเขตของงานและตั้งกลุ่มเป้าหมาย

3.  รวบรวมข้อมูลเกี่ยวกับเว็บไซต์ที่มีการบริการให้ทุกคนแลกเปลี่ยนส่งของกัน และสนใจออกแบบ

4.  ร่างโครงร่างหน้าตาแต่ละหน้า

5.  เริ่มออกแบบ Header โดยจะมี Navigation Bar อยู่ถัดจาก Logo ซึ่งจะเป็นตัวพาเราไปยังส่วนต่าง ๆ   ของเว็บไซต์ ส่วนด้านขวาจะเป็นส่วนการเข้าสู่ระบบของผู้ใช้งาน

6.  Navigation Bar จะใช้ auto layout ในการแบ่งช่องไฟ

7.  สร้าง Menu Bar สำหรับการเลือกหมวดหมู่ โดยจะสร้างกล่องเก็บข้อมูลแล้วนำมาเรียงต่อ ๆ กัน โดยใช้   auto layout

8.  ทำ Sub-Header เป็นส่วนเกริ่นในตัวของ body เพื่อบอกจุดประสงค์และเชิญชวน

9.  สร้างกล่องค้นหาอยู่ใน Sub-Header เพราะสามารถสังเกตเห็นได้ง่าย

10.  สร้าง frame บอกข้อดีของการเปิดบริการเว็บไซต์แบบนี้

11.  สร้าง frame เก็บหัวข้อและ frame รวม item ไว้ เพื่อแนะนำเสนอหนังสือที่น่าสนใจ

12.  การสร้าง item หรือรายการหนังสือ

    12.1  สร้าง frame ทำเป็นกล่อง แล้วนำรูป ชื่อ และหมวดหมู่ มาใส่ลงในกล่อง
    12.2  เพิ่ม auto layout เข้าไป ให้ง่ายต่อการใช้งาน
    12.3  จากนั้นแปลงเป็น component(แม่แบบ) แล้ว copy เพื่อนำไปใช้งาน

13.  ออกแบบส่วนสุดท้ายของหน้า Home คือ Footer เป็นที่สำหรับข้อมูลของเว็บไซต์หรือช่องการติดต่อ

14.  สร้าง frame ขึ้นมาเป็น body เก็บ History bar, Search bar, จำนวนหน้า และรายการหนังสือทั้งหมด

15.  สร้าง frame เก็บ History bar และ Search bar แล้วใช้ auto layout แบ่งช่องไฟใน History bar      และระหว่าง Search bar

16.  สร้าง frame เก็บรายการหนังสือทั้งหมดเอาไว้ โดยใช้ auto layout แบ่งช่องไฟ

17.  สร้าง frame เก็บ icon และจำนวนหน้าไว้ จากนั้นใส่ auto layout

18.  สร้าง frame เก็บปกหนังสือ เวลาโพสต์ และจำนวนครั้งที่แลกเปลี่ยน เพื่อประกอบการพิจารณา

19.  สร้าง frame เก็บข้อมูลต่าง ๆ ของหนังสือไว้ พร้อมความคิดเห็นจากผู้โพสต์ แล้วใส่ auto layout

20.  สร้างปุ่มสำหรับแลกเปลี่ยนและบันทึกเอาไว้

21.  สร้างหัวข้อ

22.  สร้าง bar สำหรับการกรอกข้อมูล

23.  สร้างปุ่มกดยืนยัน ลืมรหัสผ่าน และยังไม่มีบัญชี ให้ผู้ใช้งานได้้เลือก

24.  สร้าง frame ขึ้นมาครอบ เพื่อให้ง่ายต่อการแก้ไข

25.  สร้าง bar สำหรับชั้นหนังสือ หนังสือที่สนใจ และประวัติไว้

26.  สร้างข้อความให้เข้าสู่ระบบก่อนจะใช้บริการ

27.  สร้าง Menu ทางซ้ายขึ้นมา เป็นเหมือนแชทไว้สำหรับติดต่อขอรับหนังสือกับผู้โพสต์

28.  สร้าง frame เก็บข้อมูลต่าง ๆ ของหนังสือไว้ พร้อมความคิดเห็นจากผู้โพสต์ แล้วใส่ auto layout          ช่วยจัดตัวหนังสือ

29.  สร้าง Chat room ขึ้นมาพร้อมใส่ข้อมูลของอีกฝ่ายไว้ด้านบนของช่องแชท

30.  สร้างแป้นเปล่าไว้พิมพ์ข้อความตอบกลับกัน

31.  เมื่อทำทุกหน้าครบแล้วก็ทำการเชื่อมแต่ละหน้าเข้าหากัน
